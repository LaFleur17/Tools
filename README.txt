# SCAPCRACK 1.0

Ce script utilise principalement les outils aircrack-ng et scapy pour récupérer automatiquement les 4-way handshake.

## Prérequis

- Python 3.x / aircrack-ng / scapy / colorama / prettytable / gnome-terminal
- Deux cartes réseau configurées en mode monitor

## Utilisation

1. Assurez-vous d'avoir les prérequis installés sur votre système.
2. Connectez deux cartes réseau et assurez-vous qu'elles sont configurées en mode monitor.
3. Lancez le script en utilisant Python : `python my_script.py`.
4. Le script automatise le processus de récupération des 4-way handshake.
5. Il utilise gnome terminal pour une meilleure visibilité des actions effectuées.
